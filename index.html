Install Guide

Browsers cannot auto-move files into Minecraft folders yet, but this tool can guide and organize them.

Normal Bedrock path:

%LocalAppData%\Packages\Microsoft.MinecraftUWP_8wekyb3d8bbwe\LocalState\games\com.mojang

Put behavior packs in:

behavior_packs

Put resource packs in:

resource_packs
