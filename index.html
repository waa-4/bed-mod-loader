Bed-Mod Loader

A simple Bedrock pack helper, zip converter, and future mod generator.

Drop Packs Here

Drag .mcpack, .mcaddon, or .zip files here

Zip files can be checked and renamed into Minecraft pack formats.

Install Guide

Browsers cannot auto-move files into Minecraft folders, but this tool can check and convert valid Bedrock zips.

Normal Bedrock path:

%LocalAppData%\Packages\Microsoft.MinecraftUWP_8wekyb3d8bbwe\LocalState\games\com.mojang

Put behavior packs in:

behavior_packs

Put resource packs in:

resource_packs
