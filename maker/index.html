Bed-Mod Maker

Create simple Minecraft Bedrock add-on zips right in your browser.

Project Setup

This creates a paired Behavior Pack + Resource Pack and downloads them inside one zip.

Tip: Namespace should be simple lowercase text like derp, sibo, or hellmode. Identifiers become things like derp:black_pixel.

Item Creator

Creates BP item JSON + RP item texture data. Great for simple tools, collectibles, and crafting ingredients.

Recipe Creator

Creates shapeless or shaped crafting recipes. Use vanilla IDs like minecraft:stick or your custom IDs like derp:black_pixel.

Pattern

Structure Spawning

Upload existing .mcstructure files or create function placeholders that load structures by command.

Drop .mcstructure files here

They will be placed in BP/structures/ inside the exported zip.

To use it in-game, run something like /function spawn_structure. Your structure ID becomes namespace:file_name unless you type a full namespace:name yourself.

Function Creator

Create simple .mcfunction command files. One command per line. Do not include the slash.

Export Preview

This is what will be added to the zip. The app builds everything locally in your browser.
